📚 Apple School Manager 六大檔案說明

📦 六個檔案關聯總覽

  • Students.csv： 學生名單，每位學生一筆，包含帳號、密碼、班級等。
  • Staff.csv： 教職員名單，每位教師或行政人員一筆。
  • Classes.csv： 班級資料，綁定課程與學校。
  • Courses.csv： 課程資料，提供學科與課程對應。
  • Rosters.csv： 課程與人員的關聯（學生、老師與課程的對應表）。
  • Locations.csv： 學校地點資訊（只需一筆）。

🔗 關聯圖示

  • Students.csv ➝ 根據 Person ID 加入 Rosters.csv
  • Staff.csv ➝ 根據 Person ID 加入 Rosters.csv
  • Classes.csv ➝ 定義班級與 Courses.csv 的關聯
  • Courses.csv ➝ 課程主體，對應 Rosters.csv
  • Rosters.csv ➝ 課程與人（學生/教師）的綁定
  • Locations.csv ➝ 所有檔案中的 Location ID 對應學校代碼
